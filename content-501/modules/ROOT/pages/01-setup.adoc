= Verify OpenShift Virtualization
:navtitle: Setup

== Exercise 1: Check the Operator

[source,bash,role="execute"]
----
oc get csv -n openshift-cnv | grep -i virt
----

OpenShift Virtualization (CNV) provides KubeVirt — VMs as first-class Kubernetes resources.

== Exercise 2: Verify HyperConverged Resource

[source,bash,role="execute"]
----
oc get hyperconverged kubevirt-hyperconverged -n openshift-cnv -o jsonpath='{.status.conditions[?(@.type=="Available")].status}'
----

Should return `True` — the virtualization platform is ready.

== Exercise 3: See Both Workload Types

[source,bash,role="execute"]
----
echo "=== VMs ===" && oc get vm -n triforce
echo "=== AI Agent Pods ===" && oc get pods -n triforce -l app.kubernetes.io/part-of=triforce
----

VMs and containers — managed by the same Kubernetes control plane.
