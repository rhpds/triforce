= Run Confidential Inference
:navtitle: Inference

== Learning Objectives

* Run inference inside TDX and verify quality is unchanged
* Prove AMX acceleration works inside TDX Trust Domains

== Exercise 1: Classify a Clinical Document

. Run classification — same as the standard Triforce demo:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s -X POST {healthcare_url}/api/v1/classify \
  -H "Content-Type: application/json" \
  -d '{"text": "DISCHARGE SUMMARY: 72yo male with Type 2 Diabetes on Metformin. Recent STEMI with PCI."}' | jq .
----
+
Expected: `classification: "discharge_summary"`, `model: "granite-2b-cpu"`, latency ~800ms.

. The key insight: *same model, same latency, same quality* — but now the data is hardware-encrypted in memory. The cluster admin cannot see the patient record being processed.

== Exercise 2: Check Inference Telemetry

. Verify the inference was logged:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s {healthcare_url}/api/v1/stats | jq .
----

All inference calls are tracked — model, latency, accelerator — with the data itself remaining encrypted in the TDX Trust Domain.
