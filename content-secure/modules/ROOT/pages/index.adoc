== Welcome

AI processes your most sensitive data — patient records, financial transactions, clinical notes. During inference, that data sits in memory. Unencrypted. Visible to any cluster administrator.

*What if inference memory was hardware-encrypted?*

Intel TDX creates a Trust Domain — a hardware-enforced boundary around your AI workload. The CPU encrypts memory in silicon. No software can read the data inside.
