
Deploys the Triforce multi-agent AI platform with full IBM Kagenti governance. Demonstrates agent lifecycle management, SPIFFE zero-trust identity, MCP tool federation, and complete audit trails.

== What Gets Deployed

Same as Triforce AI, plus:

* *AgentRuntime CRDs* -- Healthcare and FinServ agents enrolled in Kagenti
* *SPIFFE Identity* -- Cryptographic workload identity per agent pod
* *MCP Gateway* -- Tool federation with governance policies
* *Audit Dashboard* -- Every inference and tool call logged and queryable
