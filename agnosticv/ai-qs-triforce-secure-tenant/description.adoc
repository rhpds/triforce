
Deploys the Triforce multi-agent AI platform with Intel TDX Confidential Containers. AI agents run inside hardware-encrypted Trust Domains — the cluster admin cannot see inference data in memory.

== What Gets Deployed

Same as Triforce AI, plus:

* *Confidential Containers* -- Healthcare and FinServ agents run with `runtimeClassName: kata-cc` inside TDX Trust Domains
* *Hardware Memory Encryption* -- Intel TDX encrypts all inference data in hardware
* *Remote Attestation* -- Trustee operator verifies genuine Intel TDX before releasing secrets
