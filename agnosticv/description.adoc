
Deploys the Triforce multi-agent AI platform on a shared base cluster via the OpenShift Sandbox API. Demonstrates polyglot agentic inference across Intel Xeon 6 CPU backends using Kagenti A2A orchestration with healthcare and financial services industry verticals.

== What Gets Deployed

* *Healthcare Agent* -- Python/FastAPI clinical NLP agent (classification, NER, summarization)
* *FinServ Agent* -- Java/Quarkus fraud scoring and compliance checking agent
* *Orchestrator* -- Go workflow coordinator with A2A agent discovery
* *Frontend Dashboard* -- React telemetry dashboard with inference logging and cost analysis
* *PostgreSQL* -- Shared state (agent registry, workflow tracking, inference log, audit trail)
* *Kafka (Redpanda)* -- Event streaming for synthetic data pipelines and agent communication
* *LiteLLM Virtual Key* -- Scoped API key for MAAS model access
