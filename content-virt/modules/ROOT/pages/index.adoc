= Triforce Virt — VMs + AI on Intel Xeon 6
:navtitle: Welcome

== The Problem

Enterprises have hundreds of VMs running databases, legacy apps, and middleware. They want to add AI — but do they need separate infrastructure?

No. Intel Xeon 6 runs both VMs and AI inference on the same server. Red Hat OpenShift Virtualization manages both workloads on one platform.

In this lab you will:

* Deploy a legacy RHEL VM alongside AI agents on the same cluster
* Prove both workloads run on the same Xeon 6 node
* Connect the VM to the AI agent via Kubernetes networking
* Show resource sharing — CPU and memory allocated across both workloads
