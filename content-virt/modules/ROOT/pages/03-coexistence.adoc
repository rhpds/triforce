= VM + AI Coexistence
:navtitle: Coexistence

== Exercise 1: Both on the Same Node

. Show all workloads — VMs and pods:
+
[source,bash,role="execute"]
----
echo "=== Virtual Machines ==="
oc get vmi -o custom-columns='NAME:.metadata.name,NODE:.status.nodeName,STATUS:.status.phase'
echo ""
echo "=== AI Agent Pods ==="
oc get pods -l 'app in (healthcare-agent,finserv-agent,orchestrator)' -o custom-columns='NAME:.metadata.name,NODE:.spec.nodeName,STATUS:.status.phase'
----

. Notice: the VM and agent pods are scheduled on the *same Xeon 6 node*. One server, two workload types.

== Exercise 2: Both Healthy

. Check the AI agent:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s {healthcare_url}/health | jq .
----

. The VM is running RHEL — verify via console:
+
[source,bash,role="execute"]
----
virtctl console legacy-database
----
+
(Use Ctrl+] to exit the console)
