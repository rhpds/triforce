== Exercise 1: Shared CPU and Memory

. View resource allocation across all workloads:
+
[source,bash,role="execute"]
----
oc adm top pods
----

. Check the node-level resource usage:
+
[source,bash,role="execute"]
----
oc adm top nodes
----

The Xeon 6 node allocates CPU and memory to both the VM and the AI agents. Kubernetes scheduler manages the sharing — no manual resource partitioning needed.

== Exercise 2: Resource Quotas

. View the namespace resource quota:
+
[source,bash,role="execute"]
----
oc get resourcequota -o yaml
----

This quota covers both VMs and containers in the same namespace — unified resource governance.
