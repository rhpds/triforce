== The Decision

[cols="2,3,3",options="header"]
|===
| When to use... | Shared (MAAS) | Self-Hosted (RHOAI)
| *Prototyping* | Yes -- instant access, many models | Overhead for exploration
| *Production* | Shared resource, queuing latency | Dedicated capacity, predictable performance
| *Data sovereignty* | Requests go to shared infra | Model runs in your namespace
| *Multi-model* | 10+ models, one endpoint | One model per InferenceService
| *Custom models* | Not supported | Full control over model weights
|===

== Next: 201 — Building an AI Agent

You have a model endpoint. A model answers questions. An *agent* solves problems.

In 201, you will build a *Red Hat + Intel Solution Architect Agent* that:

The model becomes the brain. The tools become the hands. The system prompt becomes the expertise. You deploy all three as Kubernetes pods.
