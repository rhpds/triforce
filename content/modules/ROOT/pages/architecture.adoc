== Technology Stack

[cols="1,2",options="header"]
|===
|Component |Technology

|Healthcare Agent
|Python 3.11, FastAPI, LangGraph, aiokafka, asyncpg

|FinServ Agent
|Java 21, Quarkus 3.17, SmallRye Kafka, JDBC PostgreSQL

|Orchestrator
|Go 1.22+, stdlib HTTP, A2A client

|Database
|PostgreSQL 15 with pgvector

|Event Streaming
|Redpanda (Kafka-compatible) / AMQ Streams on OCP

|Inference
|MAAS/LiteLLM proxy → Intel Xeon 6 + Gaudi 3

|Agent Protocol
|A2A v0.2.6 (JSON-RPC 2.0)

|Tool Protocol
|MCP (Model Context Protocol)

|Container Runtime
|Podman / OpenShift

|Frontend
|React 19, TypeScript, PatternFly 6, Vite
|===
