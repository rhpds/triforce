= Triforce — AI Inference at Scale on Intel Xeon 6
:navtitle: Welcome

== Welcome

Every enterprise wants AI. They've seen the demos. They've run the pilots. But when the CFO asks "what does it cost at scale?" — the room goes quiet. GPU servers cost $120K. Cloud APIs charge per token. AI is stuck in pilot because it can't afford to scale.

Triforce answers three questions:

[cols="1,3",options="header"]
|===
| Question | Answer
| Can I afford AI at scale? | Not at GPU prices. But on the CPUs you already own — yes.
| Can I run it on hardware I own? | Not if AI needs a separate GPU cluster. But on the same OpenShift, the same Xeon servers — yes.
| Can I trust it with my data? | Not if data sits unencrypted. But with TDX hardware encryption and complete audit trails — yes.
|===

== What You'll Do

In this 90-minute lab you will:

1. **Explore the architecture** — vLLM Semantic Router, three polyglot agents, MCP tool federation, AMQ Streams
2. **Run the clinical NLP pipeline** — classify, extract entities, check drug interactions, summarize — all on CPU
3. **Score financial transactions** — real LLM fraud risk assessment combined with rule-based signal detection
4. **Compare costs at scale** — Xeon 6 vs GPU servers vs cloud APIs across four volume tiers
5. **Run a live scale test** — concurrent requests, watch latency climb while cost stays flat
6. **Explore the efficiency stack** — seven engineering levers that reduce latency on the same $0 hardware
7. **See the punchline** — real numbers from your demo run, honest tradeoffs, and the optimization path

== Architecture Overview

```
vLLM Semantic Router (Red Hat)
     classify request complexity
              │
    ┌─────────┼─────────┐
    │         │         │
Healthcare  FinServ   Orchestrator
(Python)    (Quarkus)  (Go)
LangGraph   Fraud+LLM  A2A dispatch
    │         │         │
    └─────────┼─────────┘
              │
     ┌────────┼────────┐
     │        │        │
  MCP Gateway  AMQ Streams  PostgreSQL
  (tool fed.)  (streaming)  (audit log)
              │
     MAAS / LiteLLM → Intel Xeon 6 CPU
```
