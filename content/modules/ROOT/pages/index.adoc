= Triforce — AI Inference at Scale on Intel Xeon 6
:navtitle: Welcome

== What This Lab Is

A hands-on platform for running, measuring, and tuning AI inference on Intel Xeon 6 CPUs. You have a live multi-agent system deployed on OpenShift — healthcare NLP, financial fraud scoring, semantic routing, MCP tool federation — all running real LLM inference at $0/token on hardware you already own.

This is not a demo walkthrough. You will call APIs, run experiments, swap models, measure latency, and decide for yourself whether CPU inference works for your use case.

== What You Can Explore

[cols="1,3",options="header"]
|===
| Section | What You'll Do

| *Core Platform*
| Call the Healthcare and FinServ APIs directly. Run clinical NLP pipelines, score transactions, discover agents via A2A, use MCP tools — all against live services.

| *Experiments*
| Compare models head-to-head. Calculate your break-even vs cloud APIs. Fire concurrent load and watch throughput vs latency. Run the benchmark suite.

| *Optimization*
| Enable the adaptive cache and watch it learn. Try batch processing. Scale replicas and measure improvement. Explore disaggregated inference with llm-d.
|===

Every exercise produces real numbers from the deployed system. Nothing is hardcoded.
