= Triforce — AI Inference at Scale
:navtitle: Welcome

== Welcome

Triforce is a multi-agent AI platform that demonstrates enterprise inference across two industry verticals — *Healthcare* and *Financial Services* — using Intel Xeon 6 CPU, IBM Kagenti agent orchestration, and Red Hat OpenShift.

In this lab you will:

* Run a *LangGraph clinical NLP pipeline* that classifies documents, extracts medical entities, checks drug interactions, and generates summaries — entirely on Intel Xeon 6 CPU
* Score *financial transactions for fraud* using rule-based + LLM reasoning on Xeon 6
* Execute *cross-agent workflows* where a Go orchestrator dispatches tasks to Python and Java agents via the A2A protocol
* Explore *real cost comparisons* — Xeon 6 vs NVIDIA H100 vs AMD MI300X vs Claude/Gemini APIs
* Push *synthetic workloads through Kafka* and measure throughput at scale
