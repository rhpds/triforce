= Setup
:navtitle: Setup

== What You'll Do

* Log in to the OpenShift cluster
* Verify all Triforce services are running
* Confirm the APIs respond with real data

== Exercise 1: Log In

. Open the Terminal tab and log in:
+
[source,bash,role="execute",subs=attributes+]
----
oc login --insecure-skip-tls-verify -u {user} -p {password} {openshift_api_url}
----

. Switch to your project:
+
[source,bash,role="execute",subs=attributes+]
----
oc project {user}-triforce
----

== Exercise 2: Verify Services

. Check that all pods are running:
+
[source,bash,role="execute",subs=attributes+]
----
oc get pods
----
+
You should see 10 pods in `Running` status:
+
[source]
----
healthcare-agent   (x2 replicas)
finserv-agent      (x2 replicas)
orchestrator       (x1)
mcp-gateway        (x1)
semantic-router    (x1)
frontend           (x1, 2 containers)
postgres           (x1)
redpanda           (x1)
----

== Exercise 3: Test the APIs

. Healthcare agent:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s {healthcare_url}/health | jq .
----

. FinServ agent:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s {finserv_url}/health | jq .
----

. Orchestrator:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s {orchestrator_url}/health | jq .
----

. Quick pipeline test — send a clinical note and get back classification, entities, and a summary:
+
[source,bash,role="execute",subs=attributes+]
----
curl -s -X POST {healthcare_url}/api/v1/pipeline \
  -H "Content-Type: application/json" \
  -d '{"text": "Patient presents with chest pain and shortness of breath."}' \
  | jq '{classification, entity_count: (.entities | length), total_ms}'
----
+
If you see a classification, entity count, and latency — the platform is working.

== Exercise 4: Access the Frontend

Open the *Triforce Demo* tab to see the visual demo interface. You'll authenticate via OpenShift SSO on first access.

The frontend is useful for the visual pipeline animation and cost comparison charts. The lab exercises primarily use the terminal for direct API interaction.
