== The Honest Answer

GPU inference is faster. But for 80% of enterprise AI — classification, NER, fraud scoring, summarization — the question was never "is it the fastest?"

*The question is: "is it fast enough at $0?"*

The numbers you saw were real, from live API calls on Intel Xeon 6 CPU. Every model name came from the actual inference log. Every latency was measured, not hardcoded.
