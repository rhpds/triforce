= Triforce Govern — AI Agent Governance with Kagenti
:navtitle: Welcome

== Welcome

Teams are deploying AI agents across your organization. Healthcare has one. Finance built another. Operations is building a third. Each team picked their own models, tools, and deployment patterns. Nobody has a complete picture.

*What if agents were governed like any other Kubernetes resource?*

IBM Kagenti makes agents first-class CRDs — registered, discoverable, identity-verified, tool-controlled, and fully audited. Kubernetes-native governance for the agent era.

== What You'll Do

=== Required (~25 minutes)

1. **Verify the governance platform** — Kagenti operator, AgentRuntime CRDs
2. **See agents as K8s resources** — enrolled, discovered, managed
3. **Discover agents automatically** — A2A protocol, agent cards, skills
