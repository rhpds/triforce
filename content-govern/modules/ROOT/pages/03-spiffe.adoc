= Zero-Trust Identity (SPIFFE)
:navtitle: SPIFFE

== How It Works

Every agent pod receives a SPIFFE Verifiable Identity Document (SVID):

```
spiffe://triforce.local/ns/user-{guid}-triforce/sa/healthcare-agent
```

This cryptographic identity enables mTLS between agents — no static credentials.

== Exercise 1: Verify SPIFFE Identity

. Check the SPIFFE helper sidecar is running:
+
[source,bash,role="execute"]
----
oc get pods -l app=healthcare-agent -o jsonpath='{.items[0].spec.containers[*].name}'
----
+
Expected: `agent spiffe-helper`

. View the SVID certificate:
+
[source,bash,role="execute"]
----
oc exec deploy/healthcare-agent -c spiffe-helper -- cat /var/run/secrets/spiffe.io/svid.pem | openssl x509 -text -noout | head -20
----
