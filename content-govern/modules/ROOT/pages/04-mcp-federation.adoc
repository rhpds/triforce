= MCP Tool Governance
:navtitle: MCP Tools

== How It Works

The Kagenti MCP Gateway federates tools across agents. MCPServerRegistration CRDs define which tools each agent exposes, and policies control which agents can call which tools.

== Exercise 1: View Tool Registrations

. List MCP server registrations:
+
[source,bash,role="execute"]
----
oc get mcpserverregistrations
----

. Call the MCP Gateway to list available tools:
+
[source,bash,role="execute"]
----
curl -s -X POST http://mcp-gateway:8091/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":"1","method":"tools/list"}' | jq '.result.tools[].name'
----

== Exercise 2: Call a Tool Through the Gateway

. Invoke FHIR patient lookup through the gateway:
+
[source,bash,role="execute"]
----
curl -s -X POST http://mcp-gateway:8091/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":"2","method":"tools/call","params":{"name":"fhir_patient_lookup","arguments":{"patient_id":"gov-test"}}}' | jq '.result.structuredContent'
----
