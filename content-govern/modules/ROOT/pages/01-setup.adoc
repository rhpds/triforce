= Verify Kagenti Operator
:navtitle: Setup

[NOTE]
====
Kagenti is not installed on all environments. If `oc get crd | grep kagenti` returns empty, the governance features are shown in simulation mode in the frontend demo. Contact IBM for Kagenti operator access.
====

== Exercise 1: Check Kagenti CRDs

[source,bash,role="execute"]
----
oc get crd | grep -i kagenti
----

Kagenti provides `agentrimes.kagenti.io` — agents as Kubernetes resources.

== Exercise 2: Verify Agent Labels

[source,bash,role="execute"]
----
oc get pods -n triforce -l kagenti.io/type=agent -o wide
----

Agents with the `kagenti.io/type: agent` label are managed by the governance platform.

== Exercise 3: Check A2A Protocol Labels

[source,bash,role="execute"]
----
oc get pods -n triforce -l protocol.kagenti.io/a2a -o name
----

These agents participate in A2A protocol — they publish agent cards and are discoverable.
