= Verify Kagenti Operator
:navtitle: Setup

== Exercise 1: Check Kagenti Installation

. Log in and verify the Kagenti operator:
+
[source,bash,role="execute",subs=attributes+]
----
oc login --insecure-skip-tls-verify -u {user} -p {password} {openshift_api_url}
----
+
[source,bash,role="execute"]
----
oc get csv -n kagenti | grep kagenti
----

. Check the Kagenti controller is running:
+
[source,bash,role="execute"]
----
oc get pods -n kagenti
----

. Verify the CRDs are installed:
+
[source,bash,role="execute"]
----
oc get crd | grep -E "agentruntime|agentcard"
----
