= Explore: Audit Trail
:navtitle: Audit Trail

*Time: ~10 minutes* · *Role: Compliance Officer*

== What You'll Learn

Every inference call, every tool invocation, every agent decision — logged to PostgreSQL. Queryable. Exportable. Compliant.

== Exercise 1: Query the Inference Log

[source,bash,role="execute"]
----
oc exec deploy/postgres -n triforce -- psql -U triforce -d triforce -c \
  "SELECT agent_name, model, task_type, latency_ms, accelerator, created_at FROM inference_log ORDER BY created_at DESC LIMIT 10;"
----

Every LLM call tracked: which agent, which model, what task, how long, what hardware.

== Exercise 2: Aggregate by Model

[source,bash,role="execute"]
----
oc exec deploy/postgres -n triforce -- psql -U triforce -d triforce -c \
  "SELECT model, COUNT(*) as calls, ROUND(AVG(latency_ms)) as avg_ms, accelerator FROM inference_log GROUP BY model, accelerator ORDER BY calls DESC;"
----

See which models are being used most and their average latency.

== Exercise 3: Compliance Mapping

Every logged entry maps to a regulatory requirement:

[cols="1,2,2",options="header"]
|===
| Regulation | Requirement | What We Log

| *HIPAA* | Record all access to PHI data | Every inference call with input hash
| *SOX* | Audit trail for financial decisions | Fraud scores + recommendations
| *GDPR* | Data processing records | Agent, model, timestamp per request
|===

== Exercise 4: Export to SIEM

[source,bash,role="execute"]
----
oc exec deploy/postgres -n triforce -- psql -U triforce -d triforce -c \
  "COPY (SELECT * FROM inference_log ORDER BY created_at DESC LIMIT 100) TO STDOUT WITH CSV HEADER;" | head -5
----

CSV export — ready for your SIEM, compliance dashboard, or BI tool.

== The Takeaway

Full audit trail. Every AI decision recorded. Queryable by agent, model, timestamp, action. Exportable. For regulated industries, the audit trail isn't optional — and it's built into the platform, not bolted on.
