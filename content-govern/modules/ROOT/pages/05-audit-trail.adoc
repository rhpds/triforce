= Audit Trail
:navtitle: Audit

== Every Action Logged

Every inference call, every tool invocation, every workflow decision is logged to PostgreSQL.

== Exercise 1: Query the Inference Log

. View recent inference calls:
+
[source,bash,role="execute"]
----
oc exec deploy/postgres -- psql -U triforce -d triforce -c \
  "SELECT agent_name, model, task_type, latency_ms, accelerator, created_at FROM inference_log ORDER BY created_at DESC LIMIT 10;"
----

== Exercise 2: Query the Audit Trail

. View agent actions:
+
[source,bash,role="execute"]
----
oc exec deploy/postgres -- psql -U triforce -d triforce -c \
  "SELECT agent_name, action, entity_type, created_at FROM audit_trail ORDER BY created_at DESC LIMIT 10;"
----

== The Governance Value

* Every LLM call tracked: model, latency, accelerator, tokens
* Every tool call tracked: which tool, which arguments, who called it
* Every workflow tracked: which agents, what steps, what decisions
* Queryable via SQL — export to SIEM, compliance dashboards, BI tools
